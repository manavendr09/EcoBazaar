
# Ecobazar

Ecobazar is an eco-friendly e-commerce platform designed to help customers reduce their carbon footprint while shopping. The platform promotes sustainable products, encourages responsible consumption, and connects customers with environmentally conscious sellers.
With features like carbon-neutral delivery options, eco-score tracking, and green product recommendations, Ecobazar makes it easier for everyone to shop responsibly and contribute to a cleaner planet.

Ecobazar – Tech Stack Overview
Frontend

Angular (TypeScript)

Component-based architecture for maintainable UI

Reactive forms, routing, and state management

Responsive and eco-friendly UI with optimized performance

Integration with REST APIs for real-time product and carbon-footprint data

Backend

Spring Boot (Java)

RESTful API development

Secure authentication & authorization (JWT/Spring Security)

Microservice-ready architecture

Handles business logic such as carbon scoring, product management, user analytics, and order processing

Database

MySQL 8+

Stores users, orders, products, eco-score metrics, and seller data

Supports advanced indexing, JSON columns, and improved performance

Reliable for transactional e-commerce operations

Additional Tools / Optional Enhancements

JWT – Secure token-based authentication

GitHub/GitLab – Version control
